Project Ideas

Overview

The project will involve building a FAQ bot that involves speech-to-text and text-to-speech software.

Motivation:

In my current job, majority of IT support is located overseas with only hardware IT support situated locally. After the recent implementation of a new ERP system, there have been many issues and gaps of how to utilise the tool for end users. With the effects of Covid and limited resources, getting any IT support has been difficult, and responses could take up to 4 weeks.

Description of the features of the product or service:

Developing a FAQ bot could potentially answer some frequently asked questions that have simple answers. The bot could also potentially organise any urgent issues that must be prioritized. The tool would be utilised to resolve simple issues and eliminate pressure to our limited resources. This will aim to improve productivity and response times.

Tools and technologies:

FAQ bots can be created through open-source tools such as Microsoft Teams and Tidio. These user-friendly software make creating FAQ bots easy for users with little IT knowledge.

Outcome:

If the project is successful, this should reduce the number of open IT related queries and reduce the pressure to our IT team. The development should also help educate end users on commonly asked IT queries.
